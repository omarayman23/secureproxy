# Proxy Browser Design Style Guide

## Design Philosophy

### Color Palette
- **Primary Background**: Deep navy (#0a0e1a) and charcoal (#1a1d29)
- **Accent Colors**: Electric cyan (#00d4ff), mint green (#4ade80), soft purple (#8b5cf6)
- **Text Colors**: Pure white (#ffffff), light gray (#e2e8f0), medium gray (#94a3b8)
- **Warning/Alert**: Amber (#f59e0b) for caution states
- **Success/Secure**: Emerald (#10b981) for secure connections

### Typography
- **Display Font**: "Playfair Display" - Bold serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for interface elements and content
- **Code Font**: "JetBrains Mono" - Monospace for technical displays and URLs
- **Font Hierarchy**: 
  - Hero: 4rem (64px) bold
  - H1: 3rem (48px) bold
  - H2: 2rem (32px) semibold
  - H3: 1.5rem (24px) medium
  - Body: 1rem (16px) regular
  - Small: 0.875rem (14px) regular

### Visual Language
- **Minimalist Aesthetic**: Clean, uncluttered interface focusing on essential functionality
- **Cyber-Security Theme**: Subtle matrix-style patterns, geometric shapes, and glowing effects
- **Professional Trust**: Sophisticated color combinations and premium typography
- **Interactive Feedback**: Smooth animations and hover states that provide immediate user feedback
- **Privacy-First Design**: Visual cues that reinforce security and anonymity concepts

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and state transitions
- **Shader-park**: Dynamic background effects and particle systems
- **PIXI.js**: Advanced visual effects for connection visualizations
- **ECharts.js**: Privacy metrics and data visualization
- **Splide.js**: Smooth carousel for quick access websites
- **p5.js**: Creative coding for network visualization effects

### Animation & Effects
- **Text Effects**: 
  - Color cycling emphasis on key security terms
  - Split-by-letter stagger animations for headings
  - Gradient text animation for hero sections
- **Background Effects**:
  - Volumetric noise fog for depth
  - Liquid-metal displacement effects
  - Aurora gradient flow animations
- **Interactive Elements**:
  - 3D tilt effects on cards and buttons
  - Glow hover states with cyan/blue accents
  - Smooth morphing transitions between states

### Header & Navigation
- **Fixed Navigation**: Semi-transparent dark background with backdrop blur
- **Logo Area**: Custom shield icon with subtle glow effect
- **Navigation Items**: Clean typography with smooth hover transitions
- **Status Indicators**: Real-time connection status with color-coded dots

### Component Styling
- **Cards**: Dark backgrounds with subtle borders and soft shadows
- **Buttons**: Gradient backgrounds with glow effects on hover
- **Input Fields**: Dark backgrounds with cyan focus states
- **Progress Indicators**: Smooth animated bars with gradient fills
- **Modals**: Glass-morphism effect with backdrop blur

### Layout Principles
- **Grid System**: 12-column responsive grid with consistent spacing
- **Whitespace**: Generous padding and margins for breathing room
- **Hierarchy**: Clear visual hierarchy through size, color, and spacing
- **Consistency**: Unified component library throughout all pages

### Responsive Design
- **Mobile-First**: Optimized for touch interactions and small screens
- **Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- **Adaptive Typography**: Fluid scaling based on viewport size
- **Touch-Friendly**: Minimum 44px touch targets for mobile interactions
